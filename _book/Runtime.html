Runtime消息传递

[objc msgSend];

  1. 首先通过objc的isa指针找到他的class

  2. 在class的method list去找是否有msgSend

  3. 没有就去他的super class找

  4. 找到之后执行它的IMP

  5. 找到之后将key：method_name value：method_imp作为字典存储在objc_cache中

注：IMP 函数指针 记录了方法的地址

对象

struct objc_object { 

Class _Nonnull isa OBJC_ISA_AVAILABILITY; 

};
